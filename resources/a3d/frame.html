ANImAT3D

Base:
Animat3D is a proprietary 3D Web Engine based on ThreeJS. Thanks to Animat3D's API, it can be implemented in and controlled by any CMS.

3D Format:
The engine uses custom glTF 2.0 objects created with an own plugin for Autodesk® 3DS Max.

Filetypes:
Thanks to the glTF format, the generated A3D file grants the following features:
➤     Embedded animations and textures
➤     Separate files for animations and models
➤     Low file size with high resolution textures
➤     On-the-fly manipulation of 3D Skeletons
